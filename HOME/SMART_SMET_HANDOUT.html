Mastering the Art of SMART vs. SMET Goal Setting

Summary: SMART

The Mind-Centric framework. Focuses on logic, precision, and external results. Ideal for projects requiring structural discipline.

  • S: Specific - Detailed focus
  • M: Measurable - Data-driven
  • A: Achievable - Resource-based
  • R: Relevant - Objective-aligned
  • T: Time-bound - Deadline-oriented

Summary: SMET

The Embodied/Spiritual framework. Focuses on internal alignment, energy, and transformation. Ideal for lifestyle shifts.

  • S: Spiritual - Alignment with "Why"
  • M: Measurable - Qualitative growth
  • E: Embodied - Physical/Emotional resonance
  • T: Transformational - Long-term identity shift

How to build this with AI

Copy this prompt to generate your tracker: 

"Act as an Expert Performance & Wellness Coach. I want you to design a one-page Weekly Progress Monitoring Sheet for a specific goal. 

The Goal: [INSERT GOAL HERE]

Design Requirements:
Holistic Framework: Use a 'Body, Mind, and Spirit' approach. 'Mind' tracks quantitative task completion, while 'Body/Spirit' tracks energy and qualitative transformation. 

Formatting: Use a clean Markdown table for the 7-day week (Mon-Sun) or if what units make sense for the fequency. ie. If I need to standup and stretch every hour from my 8-5 desk job, a place to check each hour should be there. 

Columns Required:
  Day: (Mon-Sun)
  Task Completed: A checkbox area using LaTeX syntax $$\square$$ for a professional print look.
  Energy Levels (1-5): A column to track physical/mental vitality.
  Reflections: A column for 'Spirit' metrics with specific qualitative prompts tailored to this specific goal.

Sections to Include:
  Mindset Anchor: A powerful one-line quote or mantra related to the goal.
  Weekly Synthesis: A 'Transformational Notes' box for end-of-week review.
  Success Calculation: A formula: (Total Days / 7) * 100 with placeholders for the final percentage.
  Coach's Tip: A tailored piece of advice on how to balance effort (Mind) with recovery (Body).
  Output Style: Ensure the entire output fits on a single page. Use LaTeX for math symbols and a clean, minimalist aesthetic suitable for printing or exporting to a Word/PDF document. Make sure check boxes transfer from markdown to real boxes. When outputting pdf, make the pdf checkboxes checkable and support input fields allowing folks to use it like a form.

Flow: 
If the user has not provided the [INSERT GOAL HERE], ask them for it and anything else needed to make a nice output. 
Once they are satisfied, first ask them if you can build a pdf, html, or word document for printing.
Then ask them if they would like a standalone html page that would be interactive for the week or frequency noted.
If they say yes, create a contemporary standalone html page that would use local storage to track the data."
